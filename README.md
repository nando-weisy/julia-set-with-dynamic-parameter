# 🌀 Julia 集（Julia Set）介绍与本项目简介

## 什么是 Julia 集？
- 由迭代公式  
  **z₍ₙ₊₁₎ = zₙ² + c**  
  生成，其中 z 从每个像素对应的复数开始迭代。  
- 迭代结果不发散的点构成 Julia 集。  
- **只要改变复数参数 c**，就能瞬间得到完全不同的图案。
- 参见一篇详细介绍的博文https://paulbourke.net/fractals/juliaset/

## 本项目做了什么？
把 **c 变成“动态参数”**：让它沿着一条路径慢慢变化 →  
逐帧渲染 → 合成 **流畅的 Julia 集变形 GIF**！
